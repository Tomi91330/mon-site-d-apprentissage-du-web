Les bases du web par THOMAS LE BITOUX

Une page pour comprendre le web simplement, mais correctement

Cours suivi

Concepts clés

Partie 1

Web

Le web, c'est une immense toile mondiale.

Internet

Internet est le réseau mondial qui relie les ordinateurs entre eux.

Cloud

Le cloud permet de stocker n'importe quel fichier de manière dématérialisée.

W3C

Le W3C est une organisation qui permet de faire évoluer les technologies comme les langages client.

Partie 2

Langages client

Ce sont les langages qui s’affichent dans le navigateur, comme HTML, CSS et JavaScript.

Navigateurs

Un navigateur est un logiciel qui permet d’accéder aux sites web (Chrome, Firefox…).

Langages serveur

Ils fonctionnent en arrière-plan pour gérer les données et le fonctionnement du site.

Frameworks

Un framework est un outil qui donne une base pour coder plus vite et plus facilement.

Bases de données

Elles servent à stocker des informations comme les comptes utilisateurs ou les contenus 24 h/24 et 7 j/7.

Site responsive

Un site responsive s’adapte automatiquement à la taille de l’écran (téléphone, tablette, ordinateur).

Application mobile

Une application mobile est un programme installé sur un téléphone pour faire une tâche précise.

Partie 3

Serveurs

Un serveur est un ordinateur qui stocke les sites et les envoie quand un utilisateur les demande.

Adresse IP

C’est un numéro unique qui permet d’identifier un appareil sur Internet.

Nom d’hôte et DNS

Le DNS sert à traduire un nom de site (comme google.com) en adresse IP.

Protocoles

Les protocoles sont des règles qui permettent aux machines de communiquer entre elles (HTTP, HTTPS…).

Partie 4

Développeurs

Ils créent les sites et applications en écrivant du code dans un IDE.

Chefs de projet

Ils organisent le travail et s’assurent que le projet avance correctement.

Designers

Ils s’occupent du design et de l’expérience utilisateur.

Testeurs

Ils testent le site pour trouver les bugs et vérifier que tout fonctionne.

Techniciens

Ils installent, réparent et maintiennent le matériel informatique.

Langages

HTML

Permet de créer la structure d’une page web (titres, textes, images).

CSS

Permet de styliser le site (couleurs, tailles, mise en page).

JavaScript

Permet d’ajouter des interactions (clics, animations…).

PHP

Langage côté serveur pour rendre les sites dynamiques.

Python

Langage simple utilisé pour le web, les données et l’automatisation.

SQL

Permet de gérer et interroger les bases de données.

Outils

  • VS Code : logiciel pour écrire et tester du code
  • GitHub : permet de sauvegarder son code et travailler en équipe
  • Librairies : morceaux de code déjà prêts à être utilisés
  • Frameworks : outils complets pour développer plus vite (React, Laravel, Django)
